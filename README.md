# Random Picker

A simple web application that randomly selects an item from a list.

## Quick Start

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Built With

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui components
